What's this all about?

You probably didn’t know that the Microsoft Cloud provides support for not only Windows and .NET, but also Linux, Node.js, PHP, Java, Python, and more. Microsoft Azure is the open source friendly cloud platform designed to empower you to do more.

This app was deployed as part of the Azure DjangoCon Code Challenge.

Azure has great support for hosting Python without worrying about managing a VM and makes a great option for your Django apps.
